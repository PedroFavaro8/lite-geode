# 🚀 Geode Lite

> Performance-first fork of Geode.  
> Built to run smoothly on low-end PCs and low-end mobile devices.  
> 100% mod compatible. Zero compromises.

---

## 🎯 Mission

Geode Lite is a performance-focused fork of Geode designed to:

- Maintain **100% compatibility** with existing Geode mods
- Reduce memory usage
- Reduce CPU overhead
- Improve stability on weak hardware
- Minimize crashes on low-RAM devices
- Optimize runtime performance for mobile environments
- Deliver a smoother experience in Geometry Dash

This project exists for players whose hardware deserves respect.

---

## ⚡ Why Geode Lite?

Modern mod loaders often increase overhead, especially on:

- Low-end laptops
- Older desktop CPUs
- Entry-level Android devices
- Devices with 2GB RAM or less

Geode Lite focuses on:

- Reduced runtime overhead
- Stripped debug components in release builds
- Optimized internal loading routines
- Reduced logging in production builds
- Performance-oriented compilation flags
- Minimal background processing

No feature removal.  
No compatibility loss.  
No broken mods.

---

## 🧠 Technical Philosophy

Performance first.  
Compatibility always.  
Stability above everything.

If it slows the game down — it gets optimized.  
If it breaks compatibility — it does not ship.

---

## 🔧 Compatibility

Geode Lite aims for:

- 100% compatibility with Geode mods
- No API changes
- No required mod rewrites
- No additional developer burden

Mods built for Geode should work out-of-the-box.
